# Nicolás Palma

Desarrollador web en Rosario, Argentina.

De mañana trabajo en otra cosa. De 14 a 20 desarrollo. Con el tiempo eso se convirtió en **[Devenic](https://devenic.com.ar)** — sitios web para negocios locales con precios en pesos y sin vueltas.

Antes de meterme en el desarrollo, trabajé 14 años en IT: cableado estructurado, redes, fibra óptica, mantenimiento informático, CCTV, controles de acceso y centrales telefónicas. Eso me cambió la forma de encarar los proyectos: entiendo la infraestructura y el negocio antes de escribir una línea de código.
